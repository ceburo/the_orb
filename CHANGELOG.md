# Change Log

All notable changes to the "mia" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.1.1] - 2026-02-15

- Orb visibility improvements: increased density from 4 to 8 orbs per shell.
- Increased minimum orb size to ensure visibility when many are present.
- Optimized placement and rotation radii.

## [0.1.0] - 2026-02-15

- XP progression overhaul (10 XP for level 1, +50% per level).
- Dynamic orb size calculation to prevent overlapping.
- Automatic orb size adjustment on window resize.
- Updated `addXp` debug command.
